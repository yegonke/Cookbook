    About Us

    A short description about us is as simple as this: We are focused group of surveyors who would like to transform the world of Geomatics into a simpler and more flexible sector to enable us improve our services in areas such as:

    Survey and Mapping

    Survey is a broad field ranging from field surveys to office work. Field surveys include leveling, traversing, data capture using GPS receivers and other related outdoor activities. Office work entails plan preparations, guidance on application of land laws, quotations, mutations and other related activities done indoors.

    GIS Systems

    A GIS is a system (geographic information system) designed to capture, store, manipulate, analyze, manage, and present all types of geographical data. There are many data sources for a GIS system that include demographic data, topographical data, environment data, facilities location data, geospatial reference data and many more.

    Remote Sensing

    Remote Sesing is a susbset of GIS but is mostly alienated because of the level of tasks and skills involved in the processing of the sattelite imagery to obtain analytical maps and data. Unlike the GIS Programming, this field does not include web based coding but may involve automated programming to allow for inteligent data extraction.

    General and Fixed Surveys

    First of all, a boundary can be defined as a real or an imaginary line that separates two abutting properties. Thus, a general boundary survey is a survey done to establish a boundary of a unplanned region e.g rural areas, forests etc. On the other hand fixed boundary surveys are surveys done to determine boundaries for planned regions e.g township.

    GIS System Design and Implementation

    A GIS is a system (geographic information system) designed to capture, store, manipulate, analyze, manage, and present all types of geographical data. There are many data sources for a GIS system that include demographic data, topographical data, environment data, facilities location data, geospatial reference data and many more. All these data can be used with a GIS system.
